Want to keep track of emails sent to your customers? Easily set BCC email addresses for every WooCommerce email template individually.

== Description ==

Some webmasters may like to keep track of emails being sent to their customers. For that reason we developed WooCommerce Emails BCC which enables
you to manually receive blind copies of every WooCommerce email template. 

= Improve your Order Management =
Need email copies for accounting or sales department? Easily set a list of BCC emails for every WooCommerce email template.

== Installation ==

Just install the plugin. After that you will have a new option field "BCC" for every WooCommerce email template (see WooCommerce > Settings > Emails).
